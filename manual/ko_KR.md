# Adventure Trail Studio 사용 설명서

Adventure Trail Studio는 GPX 기반 모터사이클 투어링 코스를 만들고 편집하는 데스크톱 우선 웹 애플리케이션입니다. 이 설명서는 코스를 만들고, GPX를 가져오고, 경로를 수정하고, 다시 GPX로 내보내는 기본 사용 흐름을 안내합니다.

## 시작하기

1. 브라우저에서 앱 주소를 엽니다.
2. 카카오맵이 정상적으로 표시되는지 확인합니다.
3. 지도가 보이지 않거나 로컬 편집 모드가 표시되면 카카오 JavaScript 키와 등록된 웹 도메인을 확인합니다.

로컬 개발 기본 주소는 `http://localhost:3000`입니다. `http://127.0.0.1:3000`으로 접속한다면 카카오디벨로퍼스에도 해당 주소를 별도로 등록해야 합니다.

## 화면 구성

- 왼쪽 패널: 프로젝트, 레이어, GPX 가져오기/내보내기, 웨이포인트 목록을 관리합니다.
- 지도 영역: 경로를 그리고 수정하는 주 작업 공간입니다.
- 지도 오른쪽 툴바: 편집 도구, 역전, 나누기, 삭제 같은 주요 작업을 실행합니다.
- 오른쪽 속성 패널: 선택한 프로젝트, 구간, 포인트, 웨이포인트의 상세 정보를 확인하고 수정합니다.
- 하단 바: 현재 경로의 거리와 선택 상태를 빠르게 확인합니다.

## 프로젝트 만들기와 저장

1. 왼쪽 패널에서 새 프로젝트를 만듭니다.
2. 프로젝트 이름을 입력합니다.
3. 경로나 웨이포인트를 편집합니다.
4. 저장 버튼을 눌러 현재 프로젝트를 로컬 데이터베이스에 저장합니다.

저장된 프로젝트는 왼쪽 프로젝트 목록에서 다시 열 수 있습니다. 저장하지 않은 편집 내용이 있을 때 다른 프로젝트를 열면 현재 변경사항이 사라질 수 있습니다.

## 지도 이동과 검색

- 지도를 드래그하면 지도를 이동할 수 있습니다.
- 마우스 휠 또는 지도 컨트롤로 확대/축소할 수 있습니다.
- 상단 검색 입력창에 주소나 장소명을 입력하고 검색하면 해당 위치로 지도가 이동합니다.
- 일반 지도와 위성/하이브리드 지도를 전환할 수 있습니다.

위성 모드는 지명이 함께 보이는 하이브리드 지도 타입을 사용합니다.

## 편집 도구

지도 오른쪽 세로 툴바에서 도구를 선택합니다. 아이콘에 마우스를 올리면 도구 이름이 표시됩니다.

- 편집: 기존 포인트나 웨이포인트를 선택하고 속성을 수정합니다.
- 그리기: 지도를 클릭해 새 경로 포인트를 추가합니다. 추가한 포인트는 자동으로 선으로 연결됩니다.
- 삽입: 기존 경로 구간 사이에 포인트를 추가할 때 사용합니다.
- 연결: 서로 다른 구간의 끝점을 선택해 하나의 구간으로 병합합니다.
- 핀: 지도 위에 새 웨이포인트를 추가합니다.
- 역전: 선택한 구간의 시작점과 도착점을 반대로 바꿉니다.
- 나누기: 선택한 포인트를 기준으로 구간을 둘로 나눕니다.
- 삭제: 선택한 포인트, 구간, 웨이포인트를 삭제합니다.

기본 선택 도구는 편집입니다. 그리기, 삽입, 연결, 핀은 버튼을 직접 클릭해야 해당 기능이 적용됩니다.

## 경로 만들기

1. 툴바에서 그리기를 선택합니다.
2. 지도에서 출발 지점을 클릭합니다.
3. 이동 경로를 따라 지점을 순서대로 클릭합니다.
4. 클릭한 포인트가 빨간 원으로 표시되고 경로 라인이 연결됩니다.
5. 시작 포인트는 `S`, 종료 포인트는 `F`로 표시됩니다.

경로 라인은 기본적으로 빨간색입니다. 왼쪽 패널의 컬러칩에서 경로 색상을 변경할 수 있습니다.

## 포인트 선택과 이동

1. 편집 도구를 선택합니다.
2. 경로 위의 포인트를 클릭합니다.
3. 오른쪽 속성 패널에서 위도, 경도, 관련 정보를 확인합니다.
4. 포인트를 드래그하면 위치를 옮길 수 있습니다.

포인트를 이동해도 지도는 전체 코스를 다시 맞추기 위해 자동으로 줌아웃되지 않습니다. 현재 확대 수준을 유지한 상태로 포인트만 이동합니다.

## 오른쪽 클릭 메뉴

경로 라인 위에서 마우스 오른쪽 버튼을 클릭하면 컨텍스트 메뉴가 열립니다.

- 포인트 추가: 클릭한 경로 위치에 새 포인트를 삽입합니다.
- 분리: 클릭한 위치를 기준으로 구간을 나눕니다.
- 웨이포인트 추가: 클릭한 위치에 웨이포인트를 추가합니다.
- 삭제: 선택한 포인트를 삭제합니다.

오른쪽 클릭 메뉴는 경로 라인 근처에서 사용할 수 있습니다. 지도 빈 영역에서는 브라우저 기본 메뉴가 표시되거나 아무 동작이 없을 수 있습니다.

## 구간 나누기

1. 편집 도구로 경로 포인트를 선택합니다.
2. 툴바에서 나누기를 누릅니다.
3. 선택한 포인트를 기준으로 기존 구간이 두 개의 구간으로 분리됩니다.

구간의 시작점이나 끝점에서는 나눌 수 없습니다. 중간 포인트를 선택해야 합니다.

## 구간 연결

구간이 여러 개 있을 때 서로 다른 구간의 끝점을 연결해 하나의 구간으로 병합할 수 있습니다.

1. 툴바에서 연결을 선택합니다.
2. 첫 번째 구간의 시작점 또는 도착점을 클릭합니다.
3. 두 번째 구간의 시작점 또는 도착점을 클릭합니다.
4. 두 구간이 연결 방향에 맞게 하나의 구간으로 병합됩니다.

같은 구간의 두 포인트는 연결할 수 없습니다. 반드시 서로 다른 구간의 시작점 또는 도착점을 선택해야 합니다.

## 출발점과 도착점 역전

1. 편집 도구로 구간이나 해당 구간의 포인트를 선택합니다.
2. 툴바에서 역전을 누릅니다.
3. 선택한 구간의 포인트 순서가 반대로 바뀝니다.
4. `S`와 `F` 표시도 변경된 순서에 맞춰 갱신됩니다.

이 기능은 같은 코스를 반대 방향으로 주행할 계획을 만들 때 유용합니다.

## 웨이포인트 추가와 편집

웨이포인트는 주유, 식사, 캠핑, 위험 구간 같은 주요 지점을 표시하는 핀입니다.

웨이포인트를 추가하는 방법:

1. 툴바에서 핀을 선택합니다.
2. 새 핀 유형을 선택합니다.
3. 지도에서 웨이포인트를 추가할 위치를 클릭합니다.

경로 포인트에서 웨이포인트를 추가하는 방법:

1. 경로 포인트를 클릭합니다.
2. 오른쪽 속성 패널에서 웨이포인트 정보를 입력합니다.
3. 제목, 설명, 아이콘 유형을 저장합니다.

지원하는 유형:

- 출발
- 도착
- 주유
- 식사
- 캠핑
- 주의

웨이포인트 목록은 왼쪽 패널에서 확인할 수 있습니다. 웨이포인트를 선택하면 오른쪽 속성 패널에서 제목, 설명, 유형을 수정하거나 삭제할 수 있습니다.

## GPX 가져오기

GPX 파일을 가져오면 기존 GPX 데이터를 앱 내부 경로 모델로 변환해 지도에 표시합니다.

가져오기 방법:

1. 왼쪽 패널의 GPX 업로드 영역을 찾습니다.
2. `.gpx` 파일을 업로드 영역에 드래그 앤 드롭하거나 파일 선택 버튼을 사용합니다.
3. 가져오기가 완료되면 구간, 포인트, 웨이포인트 수가 표시됩니다.

지원하는 GPX 데이터:

- Track
- Route
- Waypoint

잘못된 좌표는 가져오기 과정에서 제외됩니다. GPX 안에 경로나 웨이포인트가 없으면 가져오기에 실패할 수 있습니다.

## GPX 내보내기

현재 프로젝트를 GPX 파일로 내보낼 수 있습니다.

1. 경로와 웨이포인트를 편집합니다.
2. 왼쪽 패널의 GPX 내보내기를 누릅니다.
3. 브라우저가 `.gpx` 파일을 다운로드합니다.

내보내기는 GPX Track 형식으로 생성됩니다. 웨이포인트가 있으면 함께 포함됩니다.

## Undo와 Redo

다음 단축키로 편집 작업을 되돌리거나 다시 적용할 수 있습니다.

- Undo: `Ctrl+Z` 또는 `Cmd+Z`
- Redo: `Ctrl+Shift+Z` 또는 `Cmd+Shift+Z`

지원하는 작업:

- 포인트 이동
- 포인트 삽입
- 포인트 삭제
- 구간 분리
- 구간 연결
- 구간 삭제
- 웨이포인트 편집

## 대용량 GPX 편집 팁

Adventure Trail Studio는 많은 포인트를 포함한 GPX도 부드럽게 다루기 위해 일부 표시를 단순화합니다.

- 직선에 가까운 구간에서는 중간 포인트 핸들이 생략될 수 있습니다.
- 방향이 바뀌는 지점 위주로 포인트 핸들이 표시됩니다.
- 원본 포인트 데이터는 유지되며, 렌더링만 단순화될 수 있습니다.
- 세밀한 수정을 하려면 지도를 확대하거나 구간을 나누어 작업하세요.

## 문제 해결

카카오맵이 표시되지 않는 경우:

1. `.env`에 `KAKAO_API_KEY`가 설정되어 있는지 확인합니다.
2. 카카오디벨로퍼스에서 JavaScript 키를 사용했는지 확인합니다.
3. 카카오디벨로퍼스의 JavaScript SDK 도메인에 현재 접속 주소가 등록되어 있는지 확인합니다.
4. 로컬 주소가 `localhost`인지 `127.0.0.1`인지 확인합니다. 두 주소는 서로 다른 출처로 취급됩니다.

주소 검색이 동작하지 않는 경우:

1. Kakao Maps SDK가 정상적으로 연결되었는지 확인합니다.
2. JavaScript 키와 도메인 설정을 다시 확인합니다.
3. 검색어를 더 구체적인 주소나 장소명으로 입력합니다.

GPX 파일을 가져오지 못하는 경우:

1. 파일 확장자가 `.gpx`인지 확인합니다.
2. GPX 안에 track, route, waypoint 중 하나 이상의 데이터가 있는지 확인합니다.
3. 좌표가 올바른 위도, 경도 형식인지 확인합니다.

## 현재 MVP 범위

이 앱은 내비게이션 앱이 아니라 GPX 편집 스튜디오입니다. 현재 MVP에서는 로컬 프로젝트 저장, GPX 가져오기/내보내기, 지도 기반 경로 편집에 집중합니다.

MVP에 포함되지 않는 기능:

- 사용자 계정
- 클라우드 동기화
- 팀 협업
- 공개 공유
- 모바일 최적화
- 오프라인 지도 타일
- 고도 그래프
- AI 라우팅
